Project FADRA:

What is project FADRA?

Project FADRA is shorted for Five A Day Run Away, it is a game made in the unity engine where you the player play as a fruit trying to escape the chef before you get eaten

The Object of Project FADRA At The Moment

Because at the moment the application is still in a prototype phase the current objective is to find eight hidden red orbs. This is because I wanted to show the main feature off which is the kitchen
